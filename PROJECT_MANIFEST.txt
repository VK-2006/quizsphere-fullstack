QuizSphere Complete Full-Stack Source
=====================================

This is a complete project archive, not a partial patch.

Core modules:
- React/Vite frontend
- Spring Boot backend
- MySQL schema and migration
- JWT authentication and authorization
- Google Sign-In
- User profile
- Quiz attempt/result/review/history
- Admin dashboard and management APIs
- Light/dark theme
- Security-question + recovery-code password reset
- Docker, Railway, Vercel, Postman, and setup documentation

Obsolete email OTP implementations are not included.
No real API keys, database passwords, Gmail App Passwords, Resend keys, Brevo keys, or JWT secrets are included.
